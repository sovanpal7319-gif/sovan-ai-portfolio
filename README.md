## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/sovanpal7319/sovan-ai-portfolio.git
```

Move into the project

```bash
cd sovan-ai-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
